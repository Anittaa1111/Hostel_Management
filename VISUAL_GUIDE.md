## Color Scheme

### Admin Panel
- **Primary**: Purple-Blue Gradient (#667eea to #764ba2)
- **Success**: Green (#28a745)
- **Danger**: Red (#dc3545)
- **Warning**: Yellow (#ffc107)
- **Background**: White cards on gradient background

### Website
- **Background**: Dark gradient (Gray-900 to Gray-800)
- **Cards**: Glass-morphism effect (Gray-800/50 with backdrop blur)
- **Accent**: Purple-Blue Gradient
- **Text**: White with varying opacity

## Typography

### Admin Panel
- **Font**: Segoe UI, Tahoma, Geneva, Verdana, sans-serif
- **Headings**: Bold, 24-42px
- **Body**: Regular, 14-16px
- **Small**: 12-14px

### Website
- **Font**: System fonts (Tailwind default)
- **Headings**: Bold, 32-60px
- **Body**: Regular, 16-18px
- **Small**: 12-14px

## Responsive Breakpoints

- **Mobile**: < 768px (Single column)
- **Tablet**: 768px - 1024px (2 columns)
- **Desktop**: > 1024px (3 columns)

## Interactive Elements

### Admin Panel
- **Buttons**: Rounded corners, gradient background, hover lift
- **Inputs**: Border highlight on focus
- **Cards**: Subtle shadow, hover scale
- **Tables**: Row hover highlight

### Website
- **Cards**: Border glow on hover
- **Buttons**: Gradient, scale on hover
- **Images**: Scale on hover
- **Links**: Color change on hover
